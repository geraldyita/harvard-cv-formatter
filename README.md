# 📄 Harvard CV Formatter

Turn your CV into a clean, professional Harvard-style resume optimized for data roles.

---

## 🚀 How to Use (30 seconds)

1. Install the skill (see below)
2. Open Claude
3. Upload your CV
4. Type:

```
Format my CV in Harvard style for data roles
```

👉 Done. Your CV will be rewritten automatically.

---

## ⚙️ How to Install

### Easy way (recommended)

1. Download `harvard-cv-formatter.skill`
2. Open Claude
3. Go to **Settings → Skills**
4. Click **Install Skill**
5. Upload the file

---

### From GitHub (manual)

```bash
git clone https://github.com/yourusername/harvard-cv-formatter.git
cp -r harvard-cv-formatter ~/.claude/skills/user/
```

Restart Claude after installing.

# 🧠 How It Works (Simple Technical Explanation)

* Reads your CV (.pdf, .docx, .txt, etc.)
* Extracts your experience and skills
* Rewrites it using:

  * Data-focused language
  * Clear structure (Harvard format)
  * Measurable impact
* Outputs a ready-to-use CV
